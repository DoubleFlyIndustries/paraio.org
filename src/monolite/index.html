Para

A general-purpose back-end framework for the cloud.

About Para

Para was designed as a simple and modular back-end framework for object persistence and retrieval. It enables your application to store its objects to a datastore (SQL or NoSQL based) and then it automatically indexes those objects and makes them searchable.

The name "Para" means "steam" in Bulgarian. And just like steam is used to power stuff, you can use Para to power your mobile or web application back-end.
